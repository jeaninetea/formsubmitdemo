Artist Statement

Every piece I make requires a leap of faith. Each one represents a balance of feeling, memory, and process.

I rarely use drawings, but depend on a dialogue between myself, the fabrics, the threads, and the embellishments that make up my work. I use a variety of techniques, such as piecing, appliqué, machine embroidery, quilting, hand stitching, bobbin work, beading, and recycling embellishments to complete each piece.

I always hope the viewer will enjoy the work as much as I enjoy making it.

Learn!

Motion Machine Embroidery:

One 3-hour class
Skills Taught: Learn to draw and color with your sewing machine using a darning/free motion foot. Sample free motion stitches and learn to blend threads as in thread painting.

Crazy Quilting with a Curve:

One 3-hour class
Bring a bag a fabric scraps and learn to sew pieces together to form a new and unique pieced fabric that can be used in quilts, home décor, and wearables. Directions for sewing a zippered bag will be included.

Loose Thread Embroidery:

One 3-hour class
Learn to decorate ordinary fabrics using snippets of decorative threads and fabrics. We will finish this into a zippered bag.

Machine Appliqué:

One 3-hour class
Learn the basic essentials of satin stitch appliqué and explore creative options for decorating wearables, home décor items, quilts and more.

Bobbin Stitching & Couching:

One 3-hour class
Skills Taught: Learn to do decorative stitching using threads that are too heavy to pass through the eye of a needle. Sample various decorative stitch techniques that can be used on wearables, home décor, quilts and more. We will finish this into a zippered bag.
